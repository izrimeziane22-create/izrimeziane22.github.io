# Portfolio Galactique

Un portfolio moderne et interactif avec un thème de galaxie dans l'espace, utilisant HTML, CSS et JavaScript.

## Fonctionnalités

- **Page d'accueil** : Présentation avec nom et courte description
- **Section À propos** : Informations personnelles
- **Section Compétences** : Liste des compétences techniques
- **Section TCS** : Technologies, Compétences et Services
- **Section Veille** : Veille technologique
- **Section Projets** : Grille de projets avec effets hover
- **Section Contact** : Formulaire de contact
- **Design responsive** : Adapté pour ordinateur, tablette et mobile
- **Animations** : Dégradé animé, particules en arrière-plan, transitions fluides
- **Thème galactique spatial** : Image de galaxie animée en arrière-plan de la page d'accueil, étoiles animées, couleurs représentant l'espace (blanc, bleu royal, violet, rose profond), texte avec effet flicker, contenu cosmique

## Technologies

- HTML5
- CSS3 (animations, responsive design)
- JavaScript (smooth scrolling, particules)

## Utilisation

Ouvrez `index.html` dans un navigateur web moderne pour voir le portfolio.
